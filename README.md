## 🚀 运行
在执行后续步骤之前，请确保您配置好了以下环境：

- nodejs ≥ 18.19.1

然后在项目根目录执行以下步骤：

### 配置环境变量
- 创建`.env`文件，并填写你的百度地图ak
```bash
VITE_APP_BAIDU_MAP_AK=<your_baidu_map_ak>
```

### 安装依赖
```sh
npm install
```

### 编译运行
```sh
npm run dev
```

### 打包部署(可选)
```sh
npm run build
```
